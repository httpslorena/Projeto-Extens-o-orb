DESCRIÇÃO DO PROJETO

Este é um projeto acadêmico de demonstração de uma aplicação web completa (Full Stack) utilizando a estrutura **Spring Boot** com Java. O principal objetivo é aplicar os conceitos de persistência de dados através do **JPA/Hibernate** para comunicação com o **MySQL**, enquanto a interface é renderizada pelo **Thymeleaf** e a segurança é gerenciada pelo **Spring Security**.
